

autoreadme:

Generate auxiliary files from README.md.

* .htaccess     - used by apache
* .header.html  - used by apache
* .readme.html  - used by apache
* .contents     - used by apache, alternate to .htaccess
* .info         - used by tree command

Options:
  --create_header       write full header into .header.html
                        Requires IndexOptions +SuppressHTMLPreamble

  -h, --help            show this help message and exit
  --no-htaccess         Do not generate .htaccess

  --ignore [IGNORE ...]
                        List of sensitive files to deny access to
  --ignore-patterns [IGNORE_PATTERNS ...]
                        Patterns of file names to deny access to

Create ~/.config/autoreadme/ignore_list.txt for non-default ignore list.
Create ~/.config/autoreadme/ignore_htaccess.txt to append extra
                        .htaccess-only deny patterns when they match
                        existing top-level entries.

Environment Variables:

MARKDOWN_CSS  Relative Location of css file, default is /.markdown.css
	            In httpd.conf IndexStyleSheet "/.markdown.css"

MARKDOWN Program to filter data to Markdown, default is b4markdown
